# leo-lov – One Page React Website

Denne opgave er løst for advokatfirmaet Leo-Lov.
Formålet var at udvikle en responsiv one-page hjemmeside i React og TypeScript,
som følger det udleverede design.

## Funktioner
- Sticky navigation
- One-page navigation med anchors
- Responsivt layout (mobil, tablet, desktop)
- Google Maps implementeret som bonus

## Teknologier
- React
- TypeScript
- SCSS
- Vite

Projektet er udviklet som en del af undervisningen.
